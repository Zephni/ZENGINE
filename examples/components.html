ZEngineObject Components

A Component can be attached to a ZEngine object. It can dynamically control the object itself and has it's own Update method that will be called every frame before the actual object update. Certain components are defined as part of ZEngine like Transform and Sprite. Check the components list to see pre set components.

To add a component to an object or get a component from an object use the following respectively:

this.AddComponent("AComponent");
this.GetComponent("AComponent");

You can create your own components by extending the global ZEngineComponents object. The standard template for a component would something like this:

ZEngineComponents.MyComponent = function(Obj, Data){
    // Component setup
}

The Obj parameter will be the object that the component belongs to, and Data will be a parameter passed along when adding the component that can be used to customise it.
