Text example

Text is a simple as adding a Text component, and setting it's Content string.

You can do this by passing it when initialising the component, or after:

// When initialising
this.AddComponent("Text", {Content: "Hello ZEngine!"});

// After initialising
var TextObj = this.AddComponent("Text");
TextObj.Config.Content = "Hello ZEngine!";

The Text component takes the following config options:

Content: "",
Font: "sans-serif",
Color: "#111111",
FontSize: "18px"
