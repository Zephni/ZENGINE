Transform

Transform is a special component in that every object is instanciated with it automatically, and is compulsary if the object is to be draw to the canvas.

The following config is possible:

Position: [ZEngine.Canvas.width/2, ZEngine.Canvas.height/2], Offset: [0.5, 0.5], Size: [0, 0], Layer: 0

Position is a point value [X, Y]. Note that positive Y will push an object "down" the canvas.

Offset is a point value [X, Y]. 0.5 would position the object relative to itself based on it's size.

Size is a point value [X, Y]. And is the litteral pixel width/height of the object

Layer places the object in an order list to be rendered. The higher the number the later Draw will be called.

A handy method if you wish to change a size based on it's aspect ratio is AspectSize(Index, Size). "Index" should be 0 (width) or 1 (height), and "Size" is the value to set it to, then the other index will be sized based on the aspect of the origional size, example below:

this.Transform.AspectSize(0, 100);

Given a object of size [200, 100], the above would set it's new size to [100, 50].
