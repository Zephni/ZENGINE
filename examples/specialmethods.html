Special Methods

ZEngine has a few special methods which makes some awkward development very easy. Let's look at each one:

ZEngine.Wait(WaitTime, Callback)

This will wait for WaitTime milliseconds, and then call the Callback function.

ZEngine.WaitDo(WaitTime, Do = null, Callback = null)

This will wait for WaitTime milliseconds, and run Do until it is reached. After this Callback is called.

ZEngine.Every.(WaitTime, Callback)

This will call Callback every WaitTime milliseconds. If Callback returns true the interval will be cleared and will not be called again.

ZEngine.WhileDo(While, Do, Callback = null)

While While is true, Do will be called. If Callback is not null it will be called once While is false.
