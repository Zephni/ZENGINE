ZEngine.LoadScript/s

To load a script and run a callback once it has loaded you can use:

ZEngine.LoadScript("Script.js", function(){
    // Script.js has loaded
});

If you wish to pull in multiple scripts you can use:

ZEngine.LoadScript(["Script1.js", "Script2.js"], function(){
    // All scripts have loaded
});

Finally, you can initialise ZEngine with scripts to load, so the main Initialise method will only be called once all assets including scripts have loaded:

ZEngine.Initialise({Canvas: "canvas", LoadScripts: ["Script.js"]}, () => {
    // Script.js has loaded
});
