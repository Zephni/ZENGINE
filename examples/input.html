ZEngine.Input

As a test, click the canvas with different mouse button, and use the arrow keys to move the text around.

Input can be tracked by using the ZEngine.Input object. Both clicks (ClickDown, LastClickDown) and keys (KeyDown, LastKeyDown) use the same notation, but clicks have some extra properties so we will use that as an example:

Both Clicks/Keys

ZEngine.Input.ClickDown(KeyCode)

This will return true every frame that a click is down with the passed KeyCode. This works the same way for KeyDown(KeyCode).

ZEngine.Input.LastClickDown

This will return the keycode for the last click, on the frame it was triggered. This works the same way for LastKeyDown.

ZEngine.Input.ClicksDown

Is an object containing all of the current clicks that are active. This works the same way for KeysDown.

Only Clicks

ZEngine.Input.ClickDownLocation

This will return the last click down location as a [X, Y] point value

ZEngine.Input.ClickUpLocation

This will return the last click up location as a [X, Y] point value

ZEngine.Input.MouseLocation

This will return the current [X, Y] point value location of the mouse
