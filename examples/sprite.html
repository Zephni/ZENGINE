Sprite example

ZEngine objects use components to make them special, one of these is the "Sprite" component. To add a sprite component use the following code:

Obj.AddComponent("Sprite", {Src: "filename.png"});

You can display just a rectangle of the source image by using the following parameters in the option list:

Obj.AddComponent("Sprite", {
    Src: "assets/RunningCat.png",
    RectX: 0,
    RectY: 0,
    RectWidth: 512,
    RectHeight: 256
});

RectX and RectY will set the begining position of the rectangle to obstract from the image.

RectWidth and RectHeight will set the width and height of the rectangle to obstract.
