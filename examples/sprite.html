Sprite / Animation example

ZEngine objects use components to make them special, one of these is the "Sprite" component. To add a sprite component to the current object you can use the AddComponent method:

this.AddComponent("Sprite", {Src: "filename.png"});

You can display just a section of the source image by using the following Rect parameters in the option list:

RectX: 0,
RectY: 0,
RectWidth: 64,
RectHeight: 64

RectX and RectY will set the begining position of the rectangle to include from the image.

RectWidth and RectHeight will set the width and height of the rectangle to include.

Note by default all of the Rect properties will default to show the entire image.

Animating is very simple with ZEngineObject's. Any sprite that has both or one of RectWidth or RectHeight is a candidate for an animation. The rect width and height should normally be set to the size of a single frame from a sprite sheet.

To animate a sprite just call the Animate method on a Sprite component. The Animate method must be passed an Animation object, which could look something like the below:

var AnAnimation = {Frames: [[0,0],[1,0],[2,0]]};

You could either pass the object in directly, or save it to the objects Data property and call as below:

this.GetComponent("Sprite").Animate(AnAnimation);

Note the sprite component would all ready have to be set before calling.

You can check this page out "examples/animation.html" to see the code running above.
