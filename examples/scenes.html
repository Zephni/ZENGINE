ZEngineScene

ZEngineScene are for handling the constructing/destructing of scenes. To create a new scene use it's constructor:

new ZEngineScene("Scene1", function(){
    // Scene ZEngine code here
});

This will automatically add the scene "Scene1" to the static ZEngineScene.Scenes list, which is retrievable by using:

ZEngineScene.Get("Scene1");

And can be run with:

ZEngineScene.Get("Scene1").Run();

Scenes can then call other scenes, once a scene is run it will "destruct" the current one, and "construct" the new one. Make sure you run your first scene within the ZEngine.Initialise method, or atleast once the DOM has loaded.
