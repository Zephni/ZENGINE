Positioning/Movement example

Positioning with ZEngineObject's is simple, as every object automatically adds a Transform component that handles all positioning for you.

Each object assigns .Transform as quick property to access the component.

There are 3 main properties in the transform component:

Position = [0, 0], Offset = [0, 0], Size = [32, 32]

All these properties are an array of two values, the first representing an X and the second the Y value.

Note Position has something a little special about it. As when calling it the position represented by the Offset will be returned. If you need get/set the position without the offset you can use position (lower case Position).

If you wanted to move an object in a single axes, you would want to use something like the following example:

this.Transform.position[0] += 1;

This would move this object along it's X axis by 1 every frame

To see how the above example is working check out the "examples/movement.html" file.
