Positioning/Movement example

Positioning with ZEngineObject's is simple, as every object automatically adds a Transform component that handles all positioning for you.

Each object assigns Transform as a quick access property to the component.

There are 3 main properties in the transform component:

this.Transform.Position = [0, 0];
this.Transform.Offset = [0, 0];
this.Transform.Size = [32, 32];

All these properties are a point value (an array of two numeric values), representing the X and Y position respectively.

To see how the above example is working check out the "examples/movement.html" file.
